# 📚 Academic Results Tracker

A modern, user-friendly web application to track and manage your academic results from 1st grade through 12th grade, plus competitive exam scores. All data is stored locally in your browser for privacy and convenience.

![Academic Results Tracker](https://img.shields.io/badge/version-1.0.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

## ✨ Features

- **School Results Tracking (1st-12th Grade)**
  - Record overall percentage/CGPA
  - Add subject-wise marks
  - Track academic years
  - Include notes and achievements

- **Competitive Exam Results**
  - Store exam scores and ranks
  - Record percentiles
  - Add section-wise breakdowns
  - Note attempt numbers and details

- **Data Management**
  - All data stored locally in your browser (privacy-focused)
  - Persistent storage across sessions
  - Easy-to-use interface
  - Delete individual results
  - View statistics dashboard

- **Clean & Modern UI**
  - Responsive design for all devices
  - Smooth animations and transitions
  - Gradient color scheme
  - Intuitive tab-based navigation

## 🛠️ Installation

### Option 1: Use GitHub Pages (Recommended)

1. Fork this repository
2. Go to Settings → Pages
3. Select "Deploy from a branch"
4. Choose `main` branch and `/ (root)` folder
5. Click Save
6. Your site will be live at `https://yourusername.github.io/academic-results-tracker/`

### Option 2: Run Locally

1. Clone the repository:
```bash
git clone https://github.com/yourusername/academic-results-tracker.git
```

2. Navigate to the project directory:
```bash
cd academic-results-tracker
```

3. Open `index.html` in your browser:
```bash
# On macOS
open index.html

# On Linux
xdg-open index.html

# On Windows
start index.html
```

Or simply drag and drop `index.html` into your browser.

## 💻 Usage

### Adding School Results

1. Click on the "School Results (1st-12th)" tab
2. Select your grade from the dropdown
3. Enter the academic year (e.g., 2020-2021)
4. Input your overall percentage or CGPA
5. (Optional) Add subject-wise marks by clicking "+ Add Subject"
6. (Optional) Include any notes or achievements
7. Click "Save School Result"

### Adding Competitive Exam Results

1. Click on the "Competitive Exams" tab
2. Enter the exam name (e.g., JEE Main, NEET, CAT)
3. Specify the exam year
4. Input your score or rank
5. (Optional) Add percentile
6. (Optional) Include section-wise scores
7. (Optional) Add additional details
8. Click "Save Exam Result"

### Viewing Results

1. Click on the "View All Results" tab
2. See statistics at the top showing total records
3. Browse through your school results (sorted by grade)
4. View competitive exam results (sorted by year)
5. Delete any result using the "Delete" button

## 🔒 Privacy & Data Storage

- **100% Local Storage**: All your data is stored in your browser's localStorage
- **No Server Required**: No data is sent to any external server
- **Privacy First**: Your academic records remain completely private
- **Data Portability**: Data stays with your browser profile

### Important Notes:
- Clearing browser data will delete your stored results
- Data is specific to the browser and device you're using
- Consider exporting your data periodically as backup (feature coming soon)
